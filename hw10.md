## 2、

```c++

void insert(int a[], int n, int value)
{
    int i;
    for (i = n ; i > 0 && value < a[i - 1]; i--)
        a[i] = a[i - 1];
    a[i] = value;
}

void printList(int *a, int n)
{
    int i;
    for (i = 0; i < n; i++)
        printf("%d  ", a[i]);
    printf("\n");
}

int main(int argc, char *argv[])
{
    int list[100] = {60, 65, 75, 80, 90, 95};
    printList(list, 6);
    printf("Insert 50: \n");
    insert(list, 6, 50);
    printList(list, 7);
     printf("Insert 67: \n");
    insert(list, 7, 67);
    printList(list, 8);
     printf("Insert 99: \n");
    insert(list, 8, 99);
    printList(list, 9);
    return 0;
}
下面是输出：
60  65  75  80  90  95
Insert 50:
50  60  65  75  80  90  95
Insert 67:
50  60  65  67  75  80  90  95
Insert 99:
50  60  65  67  75  80  90  95  99
```
选择插入点在list头上、中间、尾巴上的三个数作为算法测试的数据，因为边界容易出错，要测试头和尾；其他位置具有普遍性，只要测试一个中间的就可以了。
